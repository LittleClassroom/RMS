# CSharpAssignment

Small Windows Forms assignment targeting .NET 9.

Build and run:

```bash
dotnet build
dotnet run --project CSharpAssignment.csproj
```

Notes:
- Project was renamed from `C# Assignment.csproj` to `CSharpAssignment.csproj` to avoid tooling issues.
- SDK pinned in `global.json` to .NET 9.
